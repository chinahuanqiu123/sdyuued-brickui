# BrickUI-SDYUUED-FOR-REACT

这是山东青年政治学院信息化办公室ued团队为推动前端工程化特制的基于react的ui框架
 brickui.
  It includes::
- Babel
- CSS imports
- Image imports
- File imports
-  符合学校置标渲染条件的简单组件
-  为第三方数据源准备的高阶组件
-  页面静态化编译能力
-  一键主题定制功能
-  热更新

To get started, run `npm install` and use the `app.js` template.
